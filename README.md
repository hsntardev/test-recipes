# Test Recipes
A simple recipes website built as my very first project for learning HTML and Git/Github workflow.
## Demo
https://hsntardev.github.io/test-recipes/
## Features & What I Learned
In this project, I learned how to:
- Structure a multi-page website using HTML.
- Use absolute and relative file paths for images and links.
- Use Git and Github to track my progress and commit changes.
- Navigate the Linux command line (WSL/Ubuntu)
## Technologies Used
- HTML5
- Git & Github
